جداکننده (Dividers)

جداکننده در فایل css بصورت ::before و content تعریف می شود هم چنین میتوانید بصورت local نیز بصورت --bs-breadcrumb-divider تعریف کنید و $breadcrumb-divider را در content قرار دهید.

  • نتیجه
  • HTML
  1. خانه
  2. کتابخانه
<nav style="--bs-breadcrumb-divider: '>';" aria-label="breadcrumb">
	<ol class="breadcrumb">
		<li class="breadcrumb-item"><a href="#">Home</a></li>
		<li class="breadcrumb-item active" aria-current="page">Library</li>
	</ol>
</nav>

شما میتوانید مقدار --bs-breadcrumb-divider:' '; را خالی بگذارید.

  • نتیجه
  • HTML
  1. خانه
  2. کتابخانه
<nav style="--bs-breadcrumb-divider: '';" aria-label="breadcrumb">
	<ol class="breadcrumb">
		<li class="breadcrumb-item"><a href="#">Home</a></li>
		<li class="breadcrumb-item active" aria-current="page">Library</li>
	</ol>
</nav>
